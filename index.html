Projects

Smart Gloves for Paralyzed Individuals

The Smart Glove project is an innovative assistive technology designed to enhance communication for paralyzed patients. Utilizing a NodeMCU microcontroller, MPU6050 accelerometer, OLED display, speaker, and flex sensors/buttons, the glove interprets hand gestures to facilitate three modes of output: text on an OLED display, audio messages via a speaker, and notifications through Telegram. Additional features include fall detection, home automation for controlling lights and fans, and a medicine reminder. This comprehensive solution not only empowers patients to communicate more effectively but also enhances their safety and autonomy within their living environment. Throughout this project, I was able to learn and implement embedded C programming, significantly broadening my technical skill set and practical knowledge in developing embedded systems.
